Description

This application is intended to be the final project of the Cloud DevOps Engineer Udacity Nanodegree program. As it develops, the author will perform some steps that involve what was learned during the course, such as continuous integration, cloud computing, and deployment pipelines.

Goals

Status Description
Completed Create a simple web application
Completed Deploy web application to a Docker container
Completed Create Docker image and deploy to Docker Hub
Completed Using Jenkins to implement Continuous Integration and Continuous Deployment
Completed Attribute different labels for container versions
Completed Create Kubernetes cluster using Cloudformation
Completed Deploy Docker container to Kubernetes cluster
